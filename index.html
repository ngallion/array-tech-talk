JavaScript Arrays
and Their Functions

What is an array?

An array is a data structure
that contains a group of elements,
each identified by at least one array index, or key.

What is an array...in JavaScript?

  • JavaScript arrays are mutable, list-like objects that can contain groups of elements of one or many types
  • All arrays have length
  • Arrays have a maximum length of 4,294,967,295

Arrays sound great!
How can I get one?

  • Arrays can be initialized using the new keyword
    const someArray = new Array()
    or with the array literal
    const someOtherArray = []
